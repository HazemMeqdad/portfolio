# hazemmeqdad.com

My personal portfolio. Built with Next.js 16 and Tailwind CSS v4.

## Running it locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Editing content

All the copy (bio, work experience, skills, projects, contact info) lives in `lib/data.ts`. Change it there and every section picks it up, including the page metadata.
